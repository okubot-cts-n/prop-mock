🎯 英語学習ナビゲーション - iframe 使用例

このページは、英語学習ナビゲーションをiframeで埋め込んだ例です。

⚠️ 注意事項

GitHub Pagesでは、セキュリティヘッダーの制限によりiframeでの表示が制限される可能性があります。

確実にiframe表示するには、Netlifyなどのサービスでのホストを推奨します。

📝 iframe 埋め込みコード例

<iframe 
    src="https://okubot-cts-n.github.io/prop-mock/english_learning_navigation.html"
    title="英語学習ナビゲーション"
    width="100%"
    height="800px"
    allowfullscreen>
    お使いのブラウザはiframeをサポートしていません。
</iframe>

🎨 レスポンシブ対応のCSS例

.iframe-container {
    position: relative;
    width: 100%;
    height: 0;
    padding-bottom: 100%; /* アスペクト比 1:1 */
    overflow: hidden;
}

.iframe-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border: none;
}
