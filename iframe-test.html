🎯 iframe 表示テスト

GitHub Pages での表示テスト

⚠️ GitHub Pagesは通常、X-Frame-Options: DENYヘッダーを設定しているため、iframeでの表示が制限される可能性があります。

英語学習ナビゲーション（GitHub Pages）

代替案の説明

✅ 以下の方法でiframe表示が可能になります：
  • 独自サーバーでのホスト: 独自のWebサーバーでHTMLファイルをホスト
  • CDNの利用: Netlify、Vercel、Firebase Hostingなどのサービス
  • プロキシサーバー: ヘッダーを変更するプロキシサーバーを経由

推奨解決策

Netlifyでのホストが最も簡単で確実です：

  1. GitHubリポジトリをNetlifyに接続
  2. Netlifyが自動的にヘッダーを調整
  3. iframeでの表示が可能になる
